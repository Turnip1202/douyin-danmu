# 抖音弹幕项目重构规划文档

## 一、项目现状分析

### 1.1 当前架构问题

| 问题类别 | 具体问题 | 影响 |
|---------|---------|------|
| **文件组织结构** | 所有代码集中在根目录，缺乏层次划分 | 难以定位和维护代码 |
| **配置管理** | 配置项分散在代码中（如硬编码的URL、房间号） | 难以管理和版本控制 |
| **模块化程度** | 工具函数与业务逻辑混杂 | 代码复用性差 |
| **数据持久化** | 无数据库支持，数据仅在内存中 | 无法历史数据追溯和分析 |
| **可视化能力** | 仅命令行输出，无Web界面 | 数据查看不便，缺乏交互性 |
| **测试覆盖率** | 无单元测试和集成测试 | 重构风险高，功能验证困难 |

### 1.2 当前文件清单


## 二、重构目标与原则

### 2.1 重构目标

1. **架构清晰化**：建立分层架构，职责明确
2. **配置集中化**：统一配置管理，支持多环境
3. **模块复用化**：工具函数独立封装，提高复用率
4. **数据持久化**：引入轻量级数据库存储历史数据
5. **可视化管理**：开发Web管理界面
6. **测试覆盖化**：完善测试体系，覆盖率≥80%

### 2.2 设计原则

- **单一职责**：每个模块/类/函数只负责一个功能
- **依赖倒置**：高层模块不依赖低层模块，都依赖抽象
- **开闭原则**：对扩展开放，对修改关闭
- **接口隔离**：使用细粒度接口，避免胖接口

**签名工具示例**：

```python
# app/utils/sign_helper.py
import subprocess
import json
import os

def generate_signature(md5_value: str) -> str:
    """
    调用Node.js生成签名
    :param md5_value: MD5签名原文
    :return: 生成的签名
    """
    # 构建临时JS代码
    js_code = f"""
    // 加载签名脚本...
    var result = get_sign('{md5_value}');
    console.log(JSON.stringify(result));
    """
    
    # 写入临时文件
    temp_file = "_temp_sign.js"
    with open(temp_file, 'w', encoding='utf-8') as f:
        f.write(js_code)
    
    try:
        result = subprocess.run(
            ['node', temp_file],
            capture_output=True,
            text=True,
            encoding='utf-8',
            timeout=30
        )
        
        if result.returncode == 0:
            output = result.stdout.strip()
            lines = output.split('\n')
            signature = None
            for line in reversed(lines):
                line = line.strip()
                if line and not line.startswith('['):
                    signature = line
                    break
            
            if signature is None:
                signature = lines[-1].strip() if lines else output
            
            try:
                parsed = json.loads(signature)
                if isinstance(parsed, dict):
                    signature = parsed.get('X-Bogus') or parsed.get('signature') or signature
                else:
                    signature = parsed
            except json.JSONDecodeError:
                pass
            
            return signature
        else:
            raise Exception(f"签名生成失败: {result.stderr}")
    finally:
        if os.path.exists(temp_file):
            os.remove(temp_file)
```

**弹幕服务示例**：

```python
# app/services/danmu_service.py
import websocket
import threading
from app.utils.logger import get_logger
from app.services.message_processor import MessageProcessor

class DanmuService:
    def __init__(self, room_id: str):
        self.room_id = room_id
        self.ws = None
        self.running = False
        self.logger = get_logger(__name__)
        self.processor = MessageProcessor()
    
    def start_listening(self):
        """开始监听弹幕"""
        self.running = True
        # 构建WebSocket连接...
        threading.Thread(target=self._listen, daemon=True).start()
    
    def stop_listening(self):
        """停止监听"""
        self.running = False
        if self.ws:
            self.ws.close()
    
    def _listen(self):
        """WebSocket监听循环"""
        # 建立连接并处理消息...
        while self.running:
            try:
                # 接收消息
                message = self.ws.recv()
                # 处理消息
                self.processor.process_message(message)
            except Exception as e:
                self.logger.error(f"监听异常: {e}")
```

### 5.5 数据访问层设计

**Repository 模式**：

```python
# app/repositories/danmu_repo.py
from sqlalchemy.orm import Session
from app.models.danmu import Danmu
from datetime import datetime

class DanmuRepository:
    def __init__(self, db: Session):
        self.db = db
    
    def create(self, room_id: str, user_id: str, user_name: str, content: str, timestamp: datetime):
        """创建弹幕记录"""
        danmu = Danmu(
            room_id=room_id,
            user_id=user_id,
            user_name=user_name,
            content=content,
            timestamp=timestamp
        )
        self.db.add(danmu)
        self.db.commit()
        self.db.refresh(danmu)
        return danmu
    
    def get_by_room_id(self, room_id: str, limit: int = 100):
        """按房间ID查询弹幕"""
        return self.db.query(Danmu)\
            .filter(Danmu.room_id == room_id)\
            .order_by(Danmu.timestamp.desc())\
            .limit(limit)\
            .all()
    
    def get_by_user_id(self, user_id: str, limit: int = 50):
        """按用户ID查询弹幕"""
        return self.db.query(Danmu)\
            .filter(Danmu.user_id == user_id)\
            .order_by(Danmu.timestamp.desc())\
            .limit(limit)\
            .all()
    
    def delete_by_room_id(self, room_id: str):
        """删除指定房间的弹幕记录"""
        self.db.query(Danmu)\
            .filter(Danmu.room_id == room_id)\
            .delete()
        self.db.commit()
    
    def count_by_room_id(self, room_id: str) -> int:
        """统计房间弹幕数量"""
        return self.db.query(Danmu)\
            .filter(Danmu.room_id == room_id)\
            .count()
```

```dockerfile
CMD ["gunicorn", "-w", "4", "app.web.app:app"]
```
