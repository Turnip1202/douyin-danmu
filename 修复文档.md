# 抖音弹幕项目修复文档

## 一、问题概述

本项目是一个抖音直播弹幕监听工具，在使用过程中遇到了以下问题：

1. **依赖安装错误**：`execjs` 包无法安装
2. **模块缺失**：`google.protobuf` 模块找不到
3. **编码问题**：Windows 环境下 `UnicodeEncodeError: 'gbk' codec can't encode character`
4. **签名生成失败**：WebSocket 握手返回 400 Bad Request

---

## 二、修复过程详解

### 2.1 依赖安装问题

**问题现象**：
```bash
ERROR: Could not find a version that satisfies the requirement execjs
```

**原因分析**：
- `execjs` 不是正确的包名，正确的包名是 `PyExecJS`

**修复方案**：
```bash
# 错误命令
pip install execjs

# 正确命令
pip install PyExecJS
```

**更新 requirements.txt**：
```txt
requests
websocket-client
PyExecJS
loguru
protobuf
```

---

### 2.2 Protobuf 模块缺失

**问题现象**：
```bash
ModuleNotFoundError: No module named 'google'
```

**原因分析**：
- `douyin_pb2.py` 使用了 Protocol Buffers，需要安装 `protobuf` 库

**修复方案**：
```bash
pip install protobuf
```

---

### 2.3 Windows 编码问题（核心修复）

**问题现象**：
```bash
UnicodeEncodeError: 'gbk' codec can't encode character '\u0e11' in position 483527
```

**原因分析**：
- Windows 默认编码为 GBK，而 `execjs` 调用 Node.js 时使用 UTF-8 编码
- 编码不匹配导致中文字符无法正确处理

**修复方案**：

修改 `main.py`，添加编码设置并改用 `subprocess` 调用 Node.js：

```python
import os
import sys

# 设置 UTF-8 编码
os.environ['PYTHONIOENCODING'] = 'utf-8'

# Windows 系统控制台编码设置
if sys.platform.startswith('win'):
    try:
        sys.stdout.reconfigure(encoding='utf-8')
        sys.stderr.reconfigure(encoding='utf-8')
    except:
        pass

# 使用 subprocess 替代 execjs
import subprocess
import json

# 生成临时 JS 文件执行签名
js_code = f"""
{full_js}
var result = get_sign('{sign}');
console.log(JSON.stringify(result));
"""

with open('_temp_sign.js', 'w', encoding='utf-8') as f:
    f.write(js_code)

result = subprocess.run(
    ['node', '_temp_sign.js'],
    capture_output=True,
    text=True,
    encoding='utf-8',
    env={**os.environ, 'NODE_OPTIONS': '--no-warnings'}
)
```

### 2.4 WebSocket 握手失败

**问题现象**：
```bash
Handshake status 400 Bad Request
```

**原因分析**：
1. **参数硬编码**：`sub_room_id` 和 `user_unique_id` 是硬编码值
2. **签名值被污染**：`sign.js` 中的 `console.log` 输出混入了签名结果

**修复方案**：

#### 步骤 1：动态获取房间信息

修改 `get_room_id` 函数为 `get_room_info`，提取更多参数：

```python
def get_room_info(url):
    # ... 原有代码 ...
    
    # 提取 room_id
    match_list = re.findall(r'"roomId\\":\\"(\d+)\\",', res.text)
    if not match_list:
        match_list = re.findall(r'"room_id":(\d+)', res.text)
    room_id = match_list[0]
    
    # 提取 sub_room_id
    sub_room_match = re.findall(r'"sub_room_id":(\d+)', res.text)
    sub_room_id = sub_room_match[0] if sub_room_match else room_id
    
    # 提取 user_unique_id
    user_unique_match = re.findall(r'"user_unique_id":"?(\d+)"?', res.text)
    if not user_unique_match:
        user_unique_match = re.findall(r'"webcast_user_id":"?(\d+)"?', res.text)
    user_unique_id = user_unique_match[0] if user_unique_match else "7621938952584791552"
    
    # 提取 ttwid
    ttwid = res.cookies.get_dict().get('ttwid', '')
    
    return room_id, sub_room_id, user_unique_id, ttwid
```

#### 步骤 2：正确提取签名

过滤掉调试输出，只提取有效的签名值：

```python
if result.returncode == 0:
    output = result.stdout.strip()
    # 签名可能在最后一行（过滤调试输出）
    lines = output.split('\n')
    signature = None
    for line in reversed(lines):
        line = line.strip()
        if line and not line.startswith('[') and not line.startswith('✅') and not line.startswith('❌'):
            signature = line
            break
    
    if signature is None:
        signature = lines[-1].strip() if lines else output
```

## 五、常见问题速查

| 错误信息 | 可能原因 | 解决方案 |
|---------|---------|---------|
| `ModuleNotFoundError: google` | 缺少 protobuf | `pip install protobuf` |
| `Could not find execjs` | 包名错误 | 使用 `PyExecJS` |
| `UnicodeEncodeError: gbk` | 编码不匹配 | 使用 subprocess |
| `Handshake 400` | 参数/签名错误 | 检查 room_id、签名 |
| `Handshake 403` | IP 被封 | 更换网络或等待 |
| `签名生成失败` | Node.js 未安装 | 安装 Node.js |
